Shallow Equality คือการเปรียบเทียบความเท่ากันของ ค่า หรือ วัตถุ โดยตรวจสอบแค่ 
ระดับผิวเผิน ไม่ได้ลงลึกไปตรวจสอบค่าที่อยู่ข้างในของ object
Deep Equality หมายถึง การเปรียบเทียบ object หรือ array 
ทุกชั้น ว่ามีโครงสร้าง และค่า ที่เหมือนกันจริงๆ ไม่ใช่แค่ชั้นนอก
JSON.stringify() แปลง Object ทั้ง 2 ตัว ให้เป็น Json ด้วย JSON.stringify() 
และ เปรียบเทียบด้วย === เพื่อเช็คว่าเท่ากันหรือไม่
