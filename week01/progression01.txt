วันนี้เรียนเรื่อง array
array คือ โครงสร้างข้อมูล ที่ใช้เก็บข้อมูลหลาย ๆ ค่าไว้ในตัวแปรเดียว โดยแต่ละค่าจะถูกจัดเก็บในตำแหน่งที่เรียกว่า index
ใช้ .length เพื่อบอกจำนวนของข้อมูลใน array ได้
ใช้ .push เพื่อเอาข้อมูลไปต่อท้ายได้
ใช้ .pop ในการลบข้อมูลตำแหน่งสุดท้ายได้
ใช้ ... สามารถใช้แยกstring ออกเป็น array ได้
Array.from() สามารถcopyจาก arrray อื่นได้
